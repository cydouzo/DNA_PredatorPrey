


Requirements
====================

 * CMake (3.15+), g++-7 (C++ 11 standard)
 * Boost Library (1_70+ recommended)
 * CUDA Compatible Graphics card, adn CUDA toolkit (7.0). Tested with Ubuntu 18 on the Nvidia Titan V


Building
====================

<h3>Linux</h3>
./bootstrap.sh
cmake ./    
make

Run
====================

run/predatorPrey
